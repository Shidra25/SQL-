# SQL Repository Setup Guide

## Getting Started

This repository contains SQL tutorials, queries, and notes for learning SQL.

## How to Use This Repository

### Option 1: View Online (Easiest)
Simply visit: **https://github.com/Shidra25/SQL-**

### Option 2: Clone to Your Computer
```bash
git clone https://github.com/Shidra25/SQL-.git
cd SQL-
```

## Supported Databases

Choose one based on your preference:

### 1. **SQLite** (Recommended for Beginners)
- **Download**: https://www.sqlite.org/download.html
- **GUI Tool**: DB Browser for SQLite https://sqlitebrowser.org/
- **No installation needed** - just download and run
- **Best for**: Learning, practice, lightweight projects

### 2. **MySQL/MariaDB**
- **Download**: https://dev.mysql.com/downloads/mysql/
- **GUI Tool**: MySQL Workbench https://www.mysql.com/products/workbench/
- **Best for**: Web applications, production use

### 3. **PostgreSQL**
- **Download**: https://www.postgresql.org/download/
- **GUI Tool**: pgAdmin https://www.pgadmin.org/
- **Best for**: Advanced features, large-scale applications

## Quick Setup Steps

### For SQLite (Easiest):
1. Download DB Browser for SQLite
2. Create a new database
3. Write and execute queries
4. See examples in the `/queries` folder

### For MySQL:
1. Install MySQL Server
2. Open MySQL Workbench
3. Connect to your local database
4. Create a new schema
5. Run queries from `/queries` folder
